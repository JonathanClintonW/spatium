## Getting Started

First, prepare the npm for each folder:

```bash
cd backend
```
```bash
npm install
```

```bash
cd frontend
```
```bash
npm install next react react-dom
```
```bash
npm install
```

```bash
cd cms
```
```bash
npm install
```

In root folder install:
```bash
npm install concurrently --save-dev
```

Next, to run both backend and frontend server use:
```bash
npm start
```
